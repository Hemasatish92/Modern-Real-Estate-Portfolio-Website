## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

### Deploy

Build and deploy to GitHub Pages:

```sh
npm run deploy
```
